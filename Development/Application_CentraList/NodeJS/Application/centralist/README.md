## Setup

Scripts tested for `Ubuntu 14.04`

* To install the different required packages:

```console
bin/dependancy-installation.sh
```

* To install the different NodeJS required modules:

```console
bin/modules-update.sh
```

## Starting

* To start the NodeJS server

```console
npm start
```
